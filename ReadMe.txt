Java version - 21.0.2
Сборщик Maven version - 3.9.6

Джарник и файлы для тестирования имеются в папке out
Аргументы можно вводить в любом порядке, главное соблюдать правило, после аргументов,

 Программу постарался сделать максимально лояльной(не падающей) к ошибкам ввода.
