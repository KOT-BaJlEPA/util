Java version - 21.0.2
Сборщик Maven version - 3.9.6

Джарник и файлы для тестирования имеются в папке out
Аргументы можно вводить в любом порядке, главное соблюдать правило, после аргументов,
 за которыми ожидается значение аргумента, желательно указать значение аргумента.
 Программу постарался сделать максимально лояльной(не падающей) к ошибкам ввода.

 для теста
 Перейти в директорию где находится Джарник, запустить Джарник в cmd следующей командой
 java -jar util.jar testSource.txt testSource2.txt -s -p -p prefix  -f  wrongArgument
